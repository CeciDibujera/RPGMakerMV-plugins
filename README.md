# RPG Maker MV plugins by CeciDibujera
Here you can download the plugins I made for RPG Maker MV. **Warning**: I'm not a coding expert at all, it's possible they're not very well made. But they work and I use them, so if you want you can use them too.

## Terms of use
If you need the terms in Spanish, they're inside every plugin's help.

- You can use them for both commercial and free games.
- You can edit them as much as you like, but you must include what you did in the help.
- You must credit me as either:

    - CeciDibujera (BF Project)
    - FriKitty / CeciDibujera (BF Project)

- For some plugins I made with the help of someone else, also credit them as instructed in the help. Always read the help!

- If you look at my plugins' code just to learn how I did things and then made a different one yourself, you don't need to credit me.
